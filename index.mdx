## What is EduLearn Hub?

EduLearn Hub is a **TypeScript monorepo** that serves as the foundation for building interactive educational web applications. It brings together a production-ready backend, a type-safe database layer, auto-generated client libraries, and rich React frontends — all managed in a single pnpm workspace.

<Info>
The platform is built on [pnpm workspaces](https://pnpm.io/workspaces) with TypeScript composite projects, ensuring fast incremental builds and strong type safety across every package boundary.
</Info>

## Key features

- **Type-safe end-to-end** — TypeScript project references keep cross-package imports validated at build time
- **OpenAPI-first** — The single `openapi.yaml` spec generates both Zod validators (server-side) and React Query hooks (client-side) via Orval
- **Drizzle ORM** — Lightweight, SQL-transparent database layer with schema-first migrations
- **Express 5** — Modern async-friendly API server with structured routing and middleware
- **esbuild** — Fast production bundling producing a single CJS bundle for the API server
- **Interactive frontends** — React apps built with Tailwind CSS, Radix UI, Framer Motion, and TanStack Query
